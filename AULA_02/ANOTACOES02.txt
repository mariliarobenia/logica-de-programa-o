o que são dados? 
dados são informações que podem ser armazenadas, organizadas e utilizadas por um programa para realizar alguma tarefa.

tipos de dados:
números: ex. idade
textos: nome de pessoa
letras: 
imagens
ou qualquer informação que possa ser processada pelo computador.

variáveis: 

uma variável é o espaço na memória do computador usado para armazenar dados que podem ser alterados duarnte a execuçao de um programa.
ela funciona como uma caixa com um nome, onde guardamos informações para usar depois.
